## 本題

useStateもuseReducerも、状態管理のためのフックです。
コードを見てみましょう。

```tsx
const [name, setName] = useState<string>("");
const [age, setAge] = useState<number>(0);
```

```tsx
const [state, setState] = useReducer(
  (
   state: State,
   newState: {
    [key in keyof State]?: State[key];
   },
  ) => ({ ...state, ...newState }),
  { name: "", age: 0 }
);
```
